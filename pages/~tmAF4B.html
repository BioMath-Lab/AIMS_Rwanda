VIEWSHED ANALYSIS
Viewshed analysis is traditionally used to assess the visual impacts of a proposed development (e.g. communications tower) on the surrounding landscape. In this instance however, the sensitivity of all existing tourism infrastructure to potential visual impacts from outside the park boundary was assessed for TKR using QGIS version 3.2.3 (QGIS Development Team 2018) and 30m elevation data provided by the Shuttle Radar Topography Mission (SRTM; USGS 2014). As described by van Rooyen and van Rooyen (2017), TKRâ€™s altitudes range from 1020 MASL in the west to Drogekloof (1586 MASL) in the east. Regionally, altitudes follow a similar north-west to south-east gradient of low (907 MASL) to high (1837 MASL) elevations starting along the Kuruman River and ending atop the Langberge range (USGS 2014). A boxplot (Figure 1) of these altitudinal ranges classified by farm boundaries illustrates the distribution of higher elevations across farms Septimus, Blaaukrantz, Windhoek, Westbourne and Oatlands. Analyses were run using the advanced viewshed analysis version 0.6.0 plugin for QGIS (Cuckovic 2016) with the existing tourism infrastructure set as observer points. Results are saved as a raster grid and illustrated using the Qgis2threejs version 2.0.1 plugin for QGIS (Akagi 2013). Qgis2threejs allows us to visualize DEM and vector data in 3D on web browsers.
