BASIC INSTRUCTIONS HOW TO SETUP FOR THE COURSE

INSTALL IMPORTANT SOFTWARE
First install the necessary software below.
MapWindows GIS is needed to open the GIS project containing all the output layers from the different project outcomes while Firefox and Foxit Reader are only required if you don't already have a web broweser software (e.g. Google Chrome) or a PDF viewer (e.g. Adobe).

  • MapWindow 4.8 GIS (read the Quick Start Guide or visit https://www.mapwindow.org/ for more info)
  • Mozilla Firefox web browser (only if you don't already have a web browser)
  • Foxit Reader for PDFs (only if you don't already have a PDF reader)
