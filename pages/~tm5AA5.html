INSTRUCTIONS HOW TO USE THIS MENU

INSTALL IMPORTANT SOFTWARE
First install the necessary software below. Specifically, QGIS is needed to open the GIS project containing all the output layers from the different project outcomes while Firefox and Foxit Reader are only required if you don't already have a web broweser software (e.g. Google Chrome) or a PDF viewer (e.g. Adobe).

  • QGIS Geographic Information System version 3.2.3-Bonn
  • Mozilla Firefox web browser (only if you don't already have a web browser)
  • Foxit Reader for PDFs (only if you don't already have a PDF reader)

SIDEBAR MENU
Each menu item on the sidebar can be clicked to open the results of the different project outcomes as follows:

TSWALU KALAHARI RESERVE: Description of study area
STRATEGIC MANAGEMENT PLAN (PDF): Report prepared by Rhengu Environmental Services
SPATIAL CONSERVATION PLAN: Report prepared by BioGIS, 3D viewshed, regional assessment and GIS
REFERENCES: Literature used to compile above report
CONTACTS: Contact details for consultants Sandra MacFadyen and Ralf Kalwa
